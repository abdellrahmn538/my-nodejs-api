# My Node.js API

A simple Node.js REST API built with Express and connected to PostgreSQL for learning and practice.

## Features

- RESTful API endpoints
- CRUD operations
- Connected to PostgreSQL database
- Error handling

## Installation

1. Clone the repository:

```bash
git clone https://github.com/abdellrahmn538/my-nodejs-api.git
```

2. Install dependencies:

```bash
cd my-nodejs-api
npm install
```

3. Set up your .env file with your PostgreSQL database credentials.

4. Run the server:

```bash
node app.js
```

## Usage

Use tools like Postman to test the API endpoints.
